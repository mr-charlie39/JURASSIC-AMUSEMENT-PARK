Jurassic Amusement Park Management System
<br>
The Jurassic Amusement Park Management System is a console-based application developed in C++ using Dev-C++ and Visual Studio.
<br>
This project was built to simulate the core operational logic of an amusement park while reinforcing fundamental and intermediate C++ programming concepts.
<br>
The application uses a menu-driven structure, allowing users to interact with different functionalities in a controlled and logical flow.
<br>
It demonstrates how real-world scenarios can be translated into structured programming solutions
